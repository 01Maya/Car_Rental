# 🚗 Car Rental Website Landing Page

## 🔹 Overview
A modern car rental web application with a premium UI and smooth user experience.  
Built to showcase cars, pricing, and services with interactive animations.

---

## 🔹 Description
This project is a responsive car rental platform where users can explore vehicles and services.  
It features smooth animations, clean UI, and intuitive navigation for better UX.  
Designed with modern frontend practices focusing on performance and usability.

---

## ✨ Features
- 📱 Fully Responsive Design  
- 🎬 Smooth Animations (Framer Motion)  
- 🔗 Scroll Navigation Between Sections  
- 🖱️ Hover Effects for Interactive UI  
- ⏳ Nice Car logo Carousel

---

## 📌 Sections
- **Hero Section:** Displays main heading, CTA buttons, and branding.  
- **Features Section:** Highlights services like pricing, security, and support.  
- **Brands Section:** Shows trusted car brands.  
- **CTA Section:** Encourages users to explore or sell cars.  
- **Most Searched Cars:** Displays trending cars with details.  
- **Best Sellers:** Showcases top-selling vehicles.  
- **Why Choose Us:** Highlights unique selling points.  
- **Latest Cars:** Displays newly added vehicles.  
- **Footer:** Contains company info and contact details.  

---

## 🛠 Tech Stack
- Next.js  
- Tailwind CSS  
- Framer Motion  
- ShadCN UI
